# Concepts

หมวดนี้ใช้รวบรวมแนวคิดหลักของ Legal ERP
เพื่อให้ทีมใช้ภาษาเดียวกันก่อนลงรายละเอียดของ workflow และ module

## ควรเติมต่อ

- คำนิยามของลูกความ เรื่องงาน และคดี
- ความสัมพันธ์ระหว่างลูกความ เรื่องงาน เอกสาร และการเงิน
- กฎการตั้งชื่อและรหัสอ้างอิง
