# Getting Started

หน้านี้ใช้เป็นจุดเริ่มสำหรับทีมที่ต้องการเขียนเอกสาร Legal ERP

## รันเอกสารในเครื่อง

```bash
cd "/Users/chainarong.j/Projects/Legal ERP/Projects/Docs"
pnpm dev:website
```

เปิดเว็บที่:

```text
http://127.0.0.1:3000
```

## เพิ่มหน้าใหม่

สร้างโฟลเดอร์ใต้ `docs/app/docs` แล้วเพิ่มไฟล์ `page.mdx`

ตัวอย่าง:

```text
docs/app/docs/modules/clients/page.mdx
```

จะได้ URL:

```text
/docs/modules/clients
```

## แนวทางเขียน

- ใช้หัวข้อสั้นและตรงกับงานจริง
- เริ่มด้วยคำอธิบายว่าใครใช้หน้านี้และใช้เมื่อไร
- แยกขั้นตอนเป็นลำดับที่ทำตามได้
- ใส่ข้อควรระวังเมื่อมีผลกับข้อมูลลูกความ เอกสาร หรือการเงิน
- ถ้ายังไม่แน่ใจคำเรียก ให้บันทึกไว้ก่อนแล้วค่อยปรับใน glossary
