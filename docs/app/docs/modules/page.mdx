# Modules

หน้ากลุ่ม Modules ใช้อธิบายความสามารถและขอบเขตการทำงานของระบบ โดยไม่กำหนดราคา
แผน หรือสิทธิ์ของผู้ใช้

## Documentation Boundary

- รายละเอียดว่าระบบทำอะไรอยู่ในหน้า Module แต่ละรายการ
- แผนใดเปิดฟีเจอร์ใดให้ตรวจจาก Feature Comparison
- ผู้ใช้คนใดทำรายการได้ให้ตรวจจาก Roles และ Permissions
- ลำดับการทำงานหลักให้ตรวจจาก [Business Workflow](/docs/workflow)
