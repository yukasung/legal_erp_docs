# Legal ERP Documentation

เอกสารชุดนี้เป็นพื้นที่กลางสำหรับอธิบายการใช้งานและการดูแลระบบ Legal ERP
ตั้งแต่ภาพรวมของระบบไปจนถึงขั้นตอนทำงานจริงของแต่ละทีม

## เป้าหมายของเอกสาร

- ทำให้ผู้ใช้ใหม่เข้าใจระบบและเริ่มใช้งานได้เร็ว
- เก็บคำอธิบายของโมดูลหลัก เช่น ลูกความ เรื่องงาน เอกสาร การเงิน และรายงาน
- บันทึก workflow มาตรฐานเพื่อลดการทำงานซ้ำและลดความคลาดเคลื่อน
- เป็น reference สำหรับทีมพัฒนา ทีมปฏิบัติการ และผู้ดูแลระบบ

## โครงสร้างเริ่มต้น

เริ่มจากหน้า [Getting Started](/docs/getting-started) ก่อน แล้วค่อยเพิ่มเนื้อหา
ตามโมดูลและ workflow จริงของระบบ

## สิ่งที่ควรเขียนต่อ

- คำศัพท์หลักของ Legal ERP
- รายละเอียดแต่ละโมดูล
- ขั้นตอนงานประจำวัน
- สิทธิ์ผู้ใช้งานและบทบาท
- คำถามที่พบบ่อย
