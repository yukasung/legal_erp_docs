# Overview

Legal ERP Platform คือระบบบริหารจัดการงานกฎหมายแบบ matter-centered สำหรับ
สำนักงานกฎหมายและทีมกฎหมายในองค์กร โดยให้ `Matter` เป็น record กลางสำหรับรวม
`Client`, `Document`, `Task`, `Calendar Activity`, `Quotation`, `Billing` และ
ข้อมูลด้านการเงินที่เกี่ยวข้องไว้ในพื้นที่ทำงานเดียวกัน

เป้าหมายของระบบคือช่วยให้ทีมเห็นสถานะงานตั้งแต่รับเรื่อง สร้างเอกสาร มอบหมายงาน
นัดหมาย ออกใบเสนอราคา วางบิล รับชำระเงิน และติดตามผลการทำงานได้จากข้อมูลชุด
เดียว ลดการทำงานซ้ำและลดความคลาดเคลื่อนของข้อมูลระหว่างทีม

## Who It Serves

- สำนักงานกฎหมายที่ต้องการพื้นที่ทำงานร่วมกันสำหรับบริหารงานกฎหมาย
- ทีมกฎหมายที่ต้องดูแล `Client`, `Matter` และ `Document` จำนวนมาก
- ผู้ดูแลระบบที่ต้องจัดการ `Role`, `Permission`, `Tenant` setting และรายงาน
- ทีมการเงินหรือทีมปฏิบัติการที่ต้องติดตามสถานะ `Quotation`, `Billing` และ
  collection

## Core Capabilities

- **Client & Matter Management**: จัดการข้อมูล `Client`, `Matter`
  และทีมผู้รับผิดชอบ
- **Document Management**: สร้าง จัดเก็บ และติดตาม `Document` ที่เกี่ยวข้องกับ
  `Matter`
- **Task & Calendar**: มอบหมาย `Task`, ติดตาม deadline และจัดการ
  `Calendar Activity`
- **Quotation & Billing**: ดูแล `Quotation`, `Invoice`, `Payment` และสถานะ
  collection
- **Finance Foundation**: เตรียมข้อมูลพื้นฐานสำหรับรายงานและงานการเงินต่อเนื่อง
- **Administration**: จัดการ `Tenant`, `User`, `Role`, `Permission` และ
  configuration
