# Feature Comparison

หน้านี้แสดงฟีเจอร์ทั้งหมด 216 รายการใน 8 โมดูล เพื่อเปรียบเทียบว่าแต่ละแผน
เปิดใช้งานรายการใด ตารางนี้ใช้สำหรับตรวจสอบแผนของ Workspace ส่วนการเข้าถึง
ของสมาชิกกำหนดผ่าน Access Profile หรือสิทธิ์รายบุคคล

## Status Guide

|    สถานะ     | ความหมาย                                               |
| :----------: | ------------------------------------------------------ |
|   Included   | รวมอยู่ในแผนและเปิดใช้ได้เมื่อสมาชิกมีสิทธิ์           |
| Not Included | ไม่รวมอยู่ในแผนนั้น                                    |
|  View Only   | ดูได้ตามขอบเขต แต่ไม่สร้าง แก้ไข ลบ อนุมัติ หรือส่งออก |
|   Baseline   | ความสามารถพื้นฐานที่ต้องมีในทุกแผน                     |
|  Plan Based  | ระบบควบคุมรายการตามแผนที่ Workspace เลือก              |
|  Unlimited   | ไม่มีการจำกัดจำนวนตามรายการนั้น                        |
|     TBD      | ยังไม่อนุมัติให้จำหน่ายหรือเปิดใช้งาน                  |
