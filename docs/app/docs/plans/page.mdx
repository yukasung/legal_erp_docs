# Plans & Pricing

Legal ERP มี 5 แผนสำหรับองค์กรที่มีขนาดและรูปแบบการทำงานต่างกัน
แผนที่สูงกว่าจะรวมความสามารถของแผนก่อนหน้า และเพิ่มฟีเจอร์ตามระดับบริการ

## Plans Overview

|    แผน     |      ราคา       | เหมาะสำหรับ                                | ข้อกำหนดหลัก                                                |
| :--------: | :-------------: | ------------------------------------------ | ----------------------------------------------------------- |
|  STANDARD  |       ฟรี       | ทนายเดี่ยวหรือทีมที่เริ่มใช้งาน            | ผู้ใช้ไม่จำกัด และไม่เกิน 5 Matters ต่อเดือน                |
| ESSENTIALS |  590 บาท/เดือน  | ทีมที่ต้องการจัดตารางและติดตามงานร่วมกัน   | ผู้ใช้และ Matters ไม่จำกัด                                  |
|    PRO     | 1,250 บาท/เดือน | ทีมที่ต้องควบคุมความเสี่ยงและวัดผลงาน      | เพิ่ม Conflict Checking, Time Tracking และรายงาน            |
|  ADVANCED  | 4,200 บาท/เดือน | สำนักงานที่ต้องการ Billing, Expense และ HR | เพิ่ม Legal Billing, AR, Expense, HRM และ Payroll           |
| ENTERPRISE | 8,900 บาท/เดือน | องค์กรขนาดใหญ่ที่ต้องใช้ข้อมูลเชิงลึก      | เพิ่ม Legal Accounting และรายงานด้านการเงินและ Productivity |

## Plan Order

แผนเรียงลำดับและสืบทอดความสามารถดังนี้:

STANDARD → ESSENTIALS → PRO → ADVANCED → ENTERPRISE

การสืบทอดหมายถึงแผนถัดไปรวมความสามารถของแผนก่อนหน้า แต่ฟีเจอร์ที่มีสถานะ TBD
ยังไม่ถือว่าเปิดใช้งาน แม้อยู่ในแผนระดับสูงกว่า

## Access Model

การใช้งานฟีเจอร์ต้องผ่านการตรวจสอบสองระดับ:

1. แผนขององค์กรต้องเปิดฟีเจอร์นั้น
2. ผู้ใช้ต้องมี Role, Permission และขอบเขตข้อมูลที่อนุญาต

การซื้อแผนที่สูงขึ้นจึงไม่ทำให้ผู้ใช้ทุกคนเห็นข้อมูลหรืออนุมัติรายการได้ทันที
และการมี Permission ไม่สามารถเปิดฟีเจอร์ที่แผนขององค์กรไม่รองรับ

## Status Summary

|    สถานะ     | ผลต่อการใช้งาน                                          |
| :----------: | ------------------------------------------------------- |
|   Included   | รวมอยู่ในแผนและเปิดใช้ได้เมื่อผู้ใช้มีสิทธิ์            |
| Not Included | ไม่รวมอยู่ในแผนนั้น                                     |
|  View Only   | ดูได้ตามขอบเขต แต่ไม่สามารถเปลี่ยนแปลงหรืออนุมัติข้อมูล |
|   Baseline   | ความสามารถพื้นฐานที่ทุกแผนต้องมี                        |
|     TBD      | ยังไม่อนุมัติให้จำหน่ายหรือเปิดใช้งาน                   |
