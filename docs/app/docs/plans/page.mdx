# Plans & Pricing

Legal Practice ERP Platform มี 5 แผนสำหรับ Workspace ที่มีขนาดและรูปแบบการทำงานต่างกัน
แผนที่สูงกว่าจะรวมความสามารถของแผนก่อนหน้า และเพิ่มฟีเจอร์ตามระดับบริการ

## Plan Summary

|    แผน     | เหมาะสำหรับ                                    | ความสามารถเด่น                               |
| :--------: | ---------------------------------------------- | -------------------------------------------- |
|  STANDARD  | ผู้ใช้งานคนเดียวหรือ Workspace ที่เริ่มใช้ระบบ | ฟีเจอร์พื้นฐาน และไม่เกิน 5 Matters ต่อเดือน |
| ESSENTIALS | ทีมที่ต้องจัดตารางและติดตามงานร่วมกัน          | Matters ไม่จำกัด การจัดตารางทีม และดูรายงาน  |
|    PRO     | ทีมที่ต้องควบคุมความเสี่ยงและวัดผลงาน          | Conflict Checking, Time Tracking และรายงาน   |
|  ADVANCED  | สำนักงานที่ต้องบริหารการเงินและบุคลากร         | Billing, Expense, HR และ Payroll             |
| ENTERPRISE | Workspace ขนาดใหญ่ที่ต้องใช้ข้อมูลเชิงลึก      | Legal Accounting และรายงานระดับองค์กร        |

ดูราคาแต่ละแผนได้ที่ [Pricing](/docs/plans/pricing)

## Plan Order

STANDARD → ESSENTIALS → PRO → ADVANCED → ENTERPRISE

แผนถัดไปรวมความสามารถของแผนก่อนหน้า แต่ฟีเจอร์ที่มีสถานะ TBD
ยังไม่ถือว่าเปิดใช้งาน แม้อยู่ในแผนระดับสูงกว่า

## Module and Member Access

การเข้าใช้งานระบบพิจารณาจากสองส่วน:

1. แผนของ Workspace กำหนดว่าเปิดใช้โมดูลใดได้บ้าง
2. Workspace Admin กำหนด Access Profile หรือปรับสิทธิ์ให้สมาชิกแต่ละคน

การเปลี่ยนเป็นแผนที่สูงขึ้นจึงเป็นการเปิดความสามารถของ Workspace
แต่ไม่เปลี่ยนสิทธิ์ของสมาชิกโดยอัตโนมัติ
