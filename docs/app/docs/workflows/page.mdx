# Workflows

หมวดนี้อธิบายขั้นตอนการทำงานข้าม module
ตั้งแต่รับเรื่องใหม่จนถึงติดตามผลและปิดงาน

## Workflow เริ่มต้น

- รับเรื่องและสร้างลูกความ
- เปิด matter หรือ case
- จัดเก็บเอกสารและหลักฐาน
- ติดตามงานและกำหนดเวลา
- จัดการค่าใช้จ่ายและ billing
